# Intensity Zones & Distribution

## Polarized Training Model (Endurance)

The default intensity distribution for endurance athletes is 80/20 polarized:

- **80% below VT1 / Z2:** easy, conversational effort. This is the aerobic base.
- **20% above VT2 / threshold:** hard, purposeful work (intervals, threshold, VO2max).
- **Middle zone (tempo/Z3):** used sparingly and with specific purpose. Not banned, but not a default.

## RPE Scale (Strength)

| RPE | Description | Application |
|-----|-------------|-------------|
| 7 | Could do 3 more reps | Standard working sets in build phase |
| 8 | Could do 2 more reps | Hard working sets in build phase |
| 9 | Could do 1 more rep | Peak/test phase only |
| 10 | Maximum effort | Testing or competition only |

- RPE-based autoregulation: programmed intensity (% of 1RM or RPE target) with built-in adjustment based on daily readiness.
- Build phase default: RPE 7-8. RPE 9-10 reserved for testing or peaking.

## Cardiac Drift

HR rising at constant effort during a steady-state session. Indicates fatigue, heat stress, or dehydration.

- **Flag threshold:** > 5% drift from baseline HR at the same power/pace.
- **Significance:** suggests the session is creating more fatigue than intended, or the athlete is inadequately fueled/hydrated.
- **Action:** note in session review. If chronic, investigate fueling, heat acclimation, or base fitness.

## Recovery Time Estimation

Use these windows to estimate recovery after a session. Adjust based on athlete's recovery profile from onboarding data.

| Session Type | Recovery Window | Examples |
|-------------|----------------|----------|
| Easy/recovery | 12-24 hours | Z1-Z2 run, easy spin, light stretching |
| Moderate/tempo | 24-36 hours | Tempo run, moderate strength, Z3 work |
| Hard/threshold/VO2max | 36-48 hours | Intervals, threshold sessions, heavy lifts |
| Race or maximal effort | 48-72+ hours | Races, 1RM tests, competitive events |

## Session Type Analysis Patterns

**Endurance (run/bike/swim):**
- HR zone distribution relative to prescription
- Pace/power relative to thresholds (FTP, LTHR, CSS)
- Cardiac drift (see above)

**Strength:**
- Volume (sets x reps) vs prescription
- Load relative to benchmarks (% 1RM or RPE)
- RPE accuracy (did subjective effort match the intended load?)

**Interval sessions:**
- Work/rest ratio execution
- Power/pace consistency across intervals
- Fade pattern (did later intervals drop off significantly?)
